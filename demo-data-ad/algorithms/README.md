# 算法文档 (Algorithm Documents)

本目录存放自动驾驶算法相关的设计文档和技术报告。

## 文档类型

### 1. 感知算法 (Perception Algorithms)
- **目标检测**：车辆检测、行人检测、交通标志识别
- **车道线检测**：车道线提取、车道线跟踪、车道线预测
- **障碍物识别**：静态障碍物、动态障碍物、可行驶区域
- **语义分割**：场景理解、像素级分类、实例分割
- **3D 感知**：3D 目标检测、点云处理、深度估计
- **多传感器感知**：融合感知、传感器数据关联

### 2. 规划算法 (Planning Algorithms)
- **路径规划**：全局路径规划、A* 算法、Dijkstra 算法
- **轨迹规划**：局部轨迹规划、样条曲线、多项式轨迹
- **行为决策**：有限状态机、决策树、强化学习决策
- **运动规划**：动态规划、采样规划、优化规划
- **场景预测**：轨迹预测、意图识别、交互预测
- **路径优化**：代价函数设计、约束处理、实时优化

### 3. 控制算法 (Control Algorithms)
- **横向控制**：转向控制、车道保持、轨迹跟踪
- **纵向控制**：速度控制、加减速控制、跟车控制
- **PID 控制**：参数整定、自适应 PID、模糊 PID
- **MPC 控制**：模型预测控制、约束优化、滚动优化
- **LQR 控制**：线性二次调节器、状态反馈、最优控制
- **协同控制**：横纵向协同、多车协同、人机协同

### 4. SLAM 算法 (SLAM Algorithms)
- **视觉 SLAM**：ORB-SLAM、VINS、SVO
- **激光 SLAM**：Cartographer、LOAM、LeGO-LOAM
- **融合 SLAM**：视觉-惯性融合、激光-视觉融合
- **定位算法**：粒子滤波、卡尔曼滤波、图优化

## 元数据标签

所有文档将被标记为：
- `document_type`: `algorithm_doc`
- `algorithm_type`: `perception` | `planning` | `control` | `slam`
- `algorithm_category`: 具体算法类别（如 `object_detection`, `path_planning`）

## 文档命名规范

建议格式：`algorithm_{模块}_{算法名称}_{版本}.pdf`

示例：
- `algorithm_perception_object_detection_yolo_v3.0.pdf`
- `algorithm_planning_path_planning_astar_v2.1.pdf`
- `algorithm_control_mpc_lateral_control_v1.5.pdf`

## 文档内容要求

每个算法文档应包含：
1. **算法原理**：基本原理、数学模型、理论基础
2. **算法流程**：流程图、伪代码、关键步骤
3. **参数说明**：输入输出、可调参数、参数影响
4. **性能指标**：准确率、实时性、鲁棒性
5. **应用场景**：适用场景、限制条件、注意事项
6. **实现建议**：代码框架、优化方法、工程实践

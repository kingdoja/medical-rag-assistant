# YOLO 目标检测算法设计文档

## 算法概述

YOLO (You Only Look Once) 是一种实时目标检测算法，广泛应用于自动驾驶感知系统。

## 算法原理

### 网络架构
- **骨干网络**: Darknet-53 / CSPDarknet
- **颈部网络**: FPN + PAN
- **检测头**: 多尺度检测头

### 检测流程
1. 图像输入 (640x640)
2. 特征提取 (多尺度特征)
3. 目标检测 (边界框 + 类别)
4. 后处理 (NMS)

## 技术特点

### 优势
- **实时性**: 推理速度快 (>30 FPS)
- **准确性**: mAP 达到 50%+
- **通用性**: 支持多种目标类别

### 适用场景
- 车辆检测
- 行人检测
- 交通标志识别
- 障碍物检测

## 模型配置

### 输入配置
```yaml
input_size: [640, 640]
channels: 3
normalization: [0.485, 0.456, 0.406]
```

### 网络配置
```yaml
backbone: CSPDarknet53
neck: FPN + PAN
head: YOLOv5Head
anchors: [[10,13], [16,30], [33,23], ...]
```

### 训练配置
```yaml
batch_size: 16
learning_rate: 0.01
optimizer: SGD
epochs: 300
augmentation: mosaic, mixup, hsv
```

## 已知问题

### 小目标检测
- **问题**: 小目标检测率低
- **原因**: 特征图分辨率不足
- **解决方案**: 增加高分辨率特征图

### 遮挡场景
- **问题**: 遮挡目标漏检
- **原因**: 特征不完整
- **解决方案**: 使用部分特征检测

### 密集场景
- **问题**: 密集目标 NMS 抑制
- **原因**: IoU 阈值设置不当
- **解决方案**: 调整 NMS 参数

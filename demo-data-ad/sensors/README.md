# 传感器文档 (Sensor Documents)

本目录存放自动驾驶传感器相关的技术文档。

## 文档类型

### 1. 摄像头 (Camera)
- 规格书：分辨率、帧率、视场角、动态范围等技术参数
- 标定文档：内参标定、外参标定、畸变校正方法
- 安装指南：安装位置、角度要求、防护措施
- 维护手册：清洁保养、故障排查、更换流程

### 2. 激光雷达 (LiDAR)
- 规格书：探测距离、角分辨率、扫描频率、点云密度
- 标定文档：坐标系标定、时间同步、多雷达标定
- 技术白皮书：工作原理、性能指标、应用场景
- 对比分析：机械式 vs 固态式、不同线数对比

### 3. 毫米波雷达 (Millimeter Wave Radar)
- 规格书：探测距离、速度测量范围、角度分辨率
- 标定文档：安装角度标定、目标关联算法
- 应用指南：前向雷达、角雷达、后向雷达配置
- 性能测试：不同天气条件下的性能表现

### 4. 超声波雷达 (Ultrasonic Radar)
- 规格书：探测距离、盲区范围、响应时间
- 安装指南：安装位置、数量配置、覆盖范围
- 应用场景：泊车辅助、低速避障、盲区监测
- 故障诊断：常见故障、排查方法、更换流程

### 5. 传感器融合 (Sensor Fusion)
- 融合方案：多传感器配置方案、优势互补
- 时间同步：传感器时间戳对齐、延迟补偿
- 坐标系转换：不同传感器坐标系统一
- 数据关联：目标级融合、特征级融合

## 元数据标签

所有文档将被标记为：
- `document_type`: `sensor_doc`
- `sensor_type`: `camera` | `lidar` | `radar` | `ultrasonic`
- `content_type`: `specification` | `calibration` | `installation` | `maintenance`

## 文档命名规范

建议格式：`sensor_{类型}_{内容}_{版本}.pdf`

示例：
- `sensor_camera_spec_v1.0.pdf`
- `sensor_lidar_calibration_guide_v2.1.pdf`
- `sensor_radar_installation_manual_v1.5.pdf`

# Demo Data Workspace

本目录用于存放 `PathoMind 医疗知识与质控助手` 的首轮真实公开 demo 数据包。

## 目录用途

- `guidelines/`：指南 / 规范
- `sops/`：SOP / 流程文档
- `manuals/`：设备说明书
- `training/`：培训资料
- `sources.md`：实际下载与来源登记表

## 使用顺序

1. 先看 `docs/specs/medical-assistant/DEMO_DATA_SOURCES.md`
2. 再按来源下载资料到本目录
3. 每下载一份就同步更新 `sources.md`
4. 按 `docs/specs/medical-assistant/DEMO_SCENARIO_MAPPING.md` 跑题集映射

## 注意事项

- 这里只放实际演示数据或其登记信息
- 真源规范仍以 `docs/specs/medical-assistant/` 为准
- 禁止放入包含患者隐私的真实病例材料

# Product Brief

## 项目名称

**PathoMind 医疗知识与质控助手**

## 一句话定位

面向病理科 / 检验科内部知识管理、培训问答与流程质控场景的 AI 知识助手，支持多源文档摄取、可信检索、规范引用回答和 Agent 工作流接入。

## 为什么做这个方向

- 与现有经历闭环最强：医疗视觉检测落地经验 + RAG / Agent 工程化能力
- 与当前仓库能力最匹配：多模态摄取、混合检索、重排、MCP、评估、可观测
- 求职表达更稳：强调知识、培训、质控和流程辅助，不碰自动诊断

## 目标用户

- 病理科医生
- 检验科医生 / 技师
- 新人培训人员
- 科室质控负责人
- 院内信息化 / AI 工具接入人员

## 核心场景

1. 指南 / SOP 检索
2. 病理 / 检验术语解释
3. 规范引用回答
4. 图文混合资料检索
5. 设备 / 流程排障问答
6. 培训问答
7. MCP 接入外部 Agent

## 产品边界

### 做什么

- 知识检索
- 规范引用
- 培训支持
- 流程质控
- 工作流辅助

### 不做什么

- 自动诊断
- 高风险临床决策建议
- 替代医生判断
- 无引用依据的医疗结论

## 对仓库能力的映射

| 当前能力 | 医疗化落地方向 |
|---|---|
| BM25 + Dense + RRF + rerank | 医疗术语与规范编号的混合检索 |
| image captioning / multimodal response | 图文病例资料与培训材料 |
| trace / evaluation | 可审计、可解释、可迭代 |
| MCP tools | 医疗知识 Agent / Copilot 接入 |

## 对外叙事

推荐对外表述：

> 这是一个面向病理 / 检验科内部使用的医疗知识与质控助手，不是通用聊天机器人，也不是自动诊断系统，而是一个可被 Agent 调用的知识基础设施。

## 简历表述参考

- 将通用模块化 RAG 底座垂直改造为病理 / 检验科医疗知识与质控助手，支持指南、SOP、设备说明书和培训资料的统一检索与引用回答。
- 设计图文资料摄取链路，完成文档解析、语义切分、元数据增强、图片描述生成与增量入库。
- 实现 BM25 + Dense Retrieval + RRF 混合召回与重排，提升医疗术语、规范编号和自然语言问法并存场景下的检索准确性。
- 基于 MCP 协议封装知识检索与摘要能力，使外部 Agent / Copilot 可直接调用科室私有知识库。

# PRD

## 版本范围

- 当前版本：`v0.1`
- 当前目标：构建面向求职展示的医疗知识与质控助手最小可演示版本

## 产品目标

### 业务目标

- 降低病理 / 检验科内部资料查找成本
- 提升新人培训效率
- 支撑流程质控与规范追溯
- 形成可被 Agent 调用的医疗知识基础设施

### 求职目标

- 能在 1 分钟内说清产品定位
- 能在 3 分钟内演示核心闭环
- 能体现医疗背景 + RAG / Agent 工程能力的结合

## 用户画像

### 病理科医生

- 关注规范、案例、术语和流程依据

### 检验科技师

- 关注设备操作、异常处理、流程说明

### 新人培训人员

- 关注学习成本、问答支持、标准资料入口

### 科室质控负责人

- 关注制度一致性、规范引用和流程追溯

## 核心功能范围

### P0

- 指南 / SOP 检索
- 规范引用回答
- 图文资料统一检索
- 设备 / 流程问答
- 标准演示题集支持

### P1

- 培训 FAQ
- 多 collection 管理
- 评估看板
- MCP 工具命名医疗化

### P2

- 术语词典增强
- 主题知识地图
- Agent 编排增强

## 非目标

- 自动诊断
- 病理报告自动结论
- 临床治疗建议
- 真实患者敏感数据生产化闭环

## 核心验收口径

至少满足以下条件：

1. 能导入公开医疗资料并完成检索
2. 10 条标准演示问题中大多数返回有引用依据的结果
3. 无答案问题能给出保守或边界清晰的回应
4. Dashboard 能支撑演示链路说明
5. MCP 工具可作为后续 Agent 接入基础

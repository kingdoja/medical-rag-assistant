# 面试讲稿（1 分钟）

我把一个通用模块化 RAG + MCP 知识底座，垂直改造成了病理 / 检验科医疗知识与质控助手。它面向科室内部的指南、SOP、培训资料和设备说明书，支持图文资料摄取、混合检索、规范引用回答和 Agent 接入。

我选择这个方向，不是简单换皮，而是因为我本身有病理图像 AI 落地经验，对医疗场景下的流程、规范和资料管理痛点更敏感。相比做通用聊天机器人，我更希望把项目做成一个能真正进入科室工作流的知识工具。

技术上，这个项目基于 BM25 + Dense Retrieval + RRF + rerank 做混合检索，通过多模态摄取处理图文资料，并通过 MCP 把知识能力暴露给外部 Agent 或 Copilot 调用。同时我保留了可观测和评估能力，方便持续优化回答质量和引用可信度。
